# Linux Community Gamejolt Rules

# Rules:

Only in connection with Linux posts

Do not mention FreeBSD & Windows & Mac OS because it is not Linux

unless it is in reference to an example with Linux, for example Linux vs. FreeBSD

You are entitled to your opinions if they are expressed appropriately.

not like this: THIS THE WORST...

For security reasons, it is prohibited to post direct .iso download URLs, but you may post a link to the main page of the official website where the download is available.

PROHIBITION OF MALICIOUS OR SUSPICIOUS SITES & .ISO FILES SUCH AS Winux (Wubuntu), etc.

Rules for moderators:

The Linux News should only be related to Linux and should not have an advertising tone such as “THIS NEW LINUX IS BETTER THAN WINDOWS.” The article should be a summary with sources (if possible).
